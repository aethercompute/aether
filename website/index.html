Aether Compute

Aether is a place to train language models together.

The future of intelligence does not belong to a few companies. Not OpenAI. Not Anthropic. Not Google. Not any monopoly.

The future is built in the open. The models are free. The code is free. The work is shared by people who care.

Open source will win because it can be trusted. FOSS models will win because anyone can study them, run them, improve them, and pass them on. Doing good will win because people want tools that serve them, not tools that control them.

Aether is for that future. Many small machines can become one large effort. Many people can help train something useful. No gatekeepers. No closed weights. No permission needed.

This is not only about compute. It is about who gets to shape the next layer of human tools.

Vision

Train strong language models in public.

Keep the code open.

Keep the models open.

Let anyone help.

Let anyone learn from the result.
